    Chapter 9 Window Objects
  • In a browser, all global variables and functions are properties and methods to the window object.
  • window.location.reload() will force a reload of the page. If given the parameter of True it will force the reload form the server and not the cache.
  • Cookies are also limited to storing up to 4KB of data, and 20 cookies are allowed per domain.
  • Cookies are still useful for retaining state information (such as if a user is logged in) because they’re passed between the client and server on every HTTP request.
  • Cookies take the form of a text file that contain a list of name/value pairs separated by semicolons.
