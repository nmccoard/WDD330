    • Chapter 12 - Object-Oriented Programming in JS
  • Three main concepts: Encapsulation, Polymorphism, and Inheritance
  • Encapsulation: is keeping the inner workings hidden by putting all the programming logic inside an object and making methods available to implement the functionality.
  • Polymorphism: is the ability to use the same process can be used by different objects. With various objects being able to sharing the same method, but also letting them have the ability to override shared methods with a more specific implementation when needed.
  • Inheritance: is being able to take features form one object and then adding more properties and/or methods to it to make a new object.
